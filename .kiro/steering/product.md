# Earth Guardians - Product Overview

Interactive project portfolio viewer for Earth Guardians South America. Displays grant-funded projects in a 3D canvas with pan/zoom navigation.

## Core Features
- **Project Gallery**: Visual canvas showing projects as cards organized by status (Em Execução, Pipeline/Escrita, Concluídos)
- **Interactive Navigation**: Pan, zoom, and parallax effects on mouse movement
- **Filtering**: Filter by territory (Brasil, Equador, Bolívia) and axis (Arte, Tecnologia, Água, Direitos)
- **Project Cards**: Display title, description, KPIs, metadata, and status tags
- **Sub-project Support**: Child projects connected to parents with visual connectors

## Design Language
- Dark earth-toned palette (moss, terracotta, ink, paper)
- Stencil/industrial aesthetic with monospace fonts
- Crosshair cursor, noise texture overlays
- 3D transforms and smooth animations
